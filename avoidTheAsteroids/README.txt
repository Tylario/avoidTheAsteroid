Avoid The Asteroids
Tyler Hudson


Purpose
	Create a game using Javascript where the user moves the mouse which controls a spaceship where you must avoid the asteroids while the speed of the asteroids increases with time.

Abstraction 1
	Start button is an abstraction for the block of code that initiates everything.

Abstraction 2
	Stats button is an abstraction for the block of code that displays/alerts the top score, average live time, times played.

Algorithm 1
	Rock move algorithm that shows the rock moving left depending on time and reseting if it is at (0, y)
